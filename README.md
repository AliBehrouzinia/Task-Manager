# Task Manager App

## Description
Task Manager is a simple app to manage daily tasks. You can add, edit, delete, and mark tasks as done or undone.  
The main goal of this project is to **practice MVVM architecture, Jetpack Compose, Kotlin Flows, and Room**, while keeping **up-to-date with the Android ecosystem**.

> ⚠️ **Note:** This project is currently under development. The features and implementation may change over time as the project evolves. This README describes the initial idea behind the app.

## Features
- **Add Task**: Title, description, and due date/time
- **Edit Task**: Update task details
- **Delete Task**: Remove tasks with confirmation dialog
- **Mark Done/Undone**: Toggle task completion status
- **Task List**: Display tasks with edit/delete options
- **Local Storage**: Save tasks using Room
- **Notifications**: Remind tasks at scheduled time

## Technologies Used
- Kotlin + MVVM
- Jetpack Compose
- Kotlin Flows
- Room Database
